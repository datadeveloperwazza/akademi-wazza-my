Cara Guna UAT Ini

Jalankan test case mengikut urutan M0 → M6 kerana setiap modul bergantung pada modul sebelumnya.

Tandakan setiap test case: ✅ Lulus / ❌ Gagal (tulis nota) / ⏭ Langkau (dengan sebab).

Sebarang isu yang ditemui — log dalam jadual Log Troubleshoot di bawah.

⚙️

Persediaan Sebelum Test

Pastikan semua langkah ini selesai sebelum mula T01.

1
Deploy Code.gs versi baru
Apps Script → Deploy → New deployment → Web App → Execute as Me → Access: Anyone
2
Sahkan GAS_URL dalam index.html
URL format: https://script.google.com/macros/s/.../exec
3
Run setupOlinSheets()
Hanya sekali jika sheet baru. Mencipta 16 tab + seed 50 produk.
4
Run setupTriggers()
Apps Script → dropdown pilih "setupTriggers" → Run. Semak Triggers: mesti ada 3 (keepWarm, backupHarian, semakIntegritiHarian).
5
Isi Tetapan Sistem
• backup_folder_id — ID folder Google Drive (dari URL: .../folders/ID)
• admin_email — e-mel penerima notifikasi integriti / backup gagal
• backup_simpan_hari — 30 (atau ikut keperluan)
• void_warehouse_hari_sama — Ya atau Tidak
6
Tukar PIN admin dari 123456
Master → Staf → edit STF-01 → tukar PIN
